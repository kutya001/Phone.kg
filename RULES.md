# Tech Stack
- Фреймворк: React 19 + Vite (SPA)
- Язык: TypeScript (строгая типизация обязательна)
- Стилизация: Tailwind CSS
- Иконки: Lucide React
- Состояние: Zustand (для избранного, сравнения, UI состояния)
- Роутинг: React Router DOM
- Анимации: Framer Motion
- Форматирование: date-fns

# Architecture Rules
- Использовать только функциональные компоненты и хуки. Никаких классовых компонентов.
- Разделять логику и UI: запросы к данным (чтение JSON) выносить в сервисы/хуки.
- Все интерфейсы TypeScript хранить в `src/types/`.
- Использовать `localStorage` для хранения избранного и списка сравнения.
- Данные хранить в статических JSON файлах в папке `src/data/` и импортировать их.

# AI Guidelines
- Не использовать устаревшие библиотеки.
- Перед написанием кода проверять `SPEC.md` и `systemArchitecture.md`.
