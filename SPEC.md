# Спецификация API и UI (PhoneCatalog)

## 1. Концепция
PhoneCatalog — это агрегатор-каталог телефонов и аксессуаров. Не является магазином (покупка не происходит на сайте). Кнопка "Купить" ведёт к конкретному продавцу.
Все данные хранятся в статических JSON-файлах.

## 2. Сущности (Интерфейсы)
- **Category**: Категории товаров (смартфоны, аксессуары и т.д.)
- **Brand**: Бренды (Apple, Samsung и т.д.)
- **Product**: Товар (модель устройства) с характеристиками, рейтингом, ценами.
- **Seller**: Продавец (магазин или частное лицо) с контактами и рейтингом.
- **SellerOffer**: Предложение продавца для конкретного товара (цена, состояние).
- **ProductReview**: Отзыв пользователя о товаре.

## 3. Требования к UI
- **Шапка (Header)**: Логотип, навигация, строка поиска (с дропдауном), иконки избранного и сравнения. Адаптивное меню (гамбургер).
- **Главная страница**: Hero-баннер, сетка категорий, популярные бренды, топ смартфонов (горизонтальный скролл), топ аксессуаров.
- **Каталог**: Фильтры (категория, бренд, цена, состояние, рейтинг), сортировка, переключение вида (сетка/список), пагинация.
- **Карточка товара**: Галерея, инфоблок (цены, рейтинг), вкладки (Описание, Характеристики, Продавцы, Отзывы).
- **Блок продавцов**: Вкладки "Новые" и "Б/У", карточки предложений с контактами (WhatsApp, Telegram, Телефон).
- **Страница продавца**: Информация о продавце, список его объявлений, отзывы.
- **Поиск**: Быстрый поиск в дропдауне, страница результатов поиска.
- **Сравнение**: Таблица характеристик (максимум 3 товара).
- **Избранное**: Список сохраненных товаров (localStorage).
- **О проекте**: Информационная страница с FAQ.

## 4. Дизайн-система
- **Цвета**: Primary (Синий #2563EB), Secondary (Оранжевый #F97316), Success (Зеленый #16A34A), Warning (Желтый #D97706).
- **Шрифты**: Golos Text (заголовки), Inter (текст), JetBrains Mono (цены).
- **Адаптивность**: Mobile-first, breakpoints (sm, md, lg, xl, 2xl).
